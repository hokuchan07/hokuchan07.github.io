4 打診当日の動線

① 公庫横浜支店（5/15）

  1. 受付で「再チャレンジ支援資金のご相談」と切り出し
  2. カバーレター §03 → 経歴書 §02 → 創業計画書 §01 の順で提示
  3. 既往書類（経緯書・決算書・源泉徴収票・免許証）を補足提示
  4. v3.1資料群（事業計画書・資金計画書・資金繰り表・想定QA）を「詳細資料」として配布
  5. 5/11追送予定書類は「追って提出」と明示

② きらぼし銀行白井支店（5/15-5/17）

  1. 中村様の住宅ローン担当者経由で融資課に取次依頼
  2. カバーレター §04 → 経歴書 §02 → 事業計画書 v3.1 の順で提示
  3. 一時撤退の経緯は 先行開示（隠さない）
  4. 神奈川県信用保証協会付き融資としての申込条件を確認

③ 神奈川県内信金（中栄 or さがみ／5/17-5/21）

  1. 事業所所在地（厚木市金田）に最も近い支店へ
  2. カバーレター §05 → 経歴書 §02 → 事業計画書 v3.1
  3. 「地域貢献事業」の文脈を強調
  4. 信用保証協会付き融資としての申込条件を確認
