🔄 同期ボタンの使い方（図解）

書いたナレッジをみんなと共有するためのボタンです。押すだけ。自動同期が裏で止まっても、これで誰でも直せます。

① ボタンの場所

画面のいちばん下の青い帯の左のほう。🔄 のとなりの数字が同期ボタンです。
― エディタ（ここで作業）―
⎇ main 🔄 1 ↓   0 ↑ 福祉ナレッジ ⊘ 0 ⚠ 0
⬆ この「🔄 数字」が同期ボタン。クリックする。

↓ の数字＝みんなから受け取る件数 ／ ↑ の数字＝自分が共有する件数。
どちらかに数字が出ていたら、まだ同期できていないサインです。

② 押すとどうなる（流れ）

いま
🔄 1↓ 0↑
未同期の数字が出ている
▶
クリック
🔄 …
受け取り＋共有が走る
▶
完了
🔄 0↓ 0↑
数字が消えたら最新！

数字が 0 になれば成功です。むずかしい操作はありません。

③ 覚えるルールは3つだけ

🌅
朝、開いたら押すみんなの最新を受け取る
🏠
帰る前に押す自分の作業を共有する
🔢
数字が出たら押す溜めない・放置しない

④ こんな表示が出たら

表示意味どうする
🔄 のとなりに数字未共有／未受信あり押す
Cannot rebase onto multiple branches同期がつまずいた
（2026/6/1に対策済み）		一度開き直す。直らなければ鈴木さん／齋藤さんへ連絡
ぐるぐる回って終わらないネット不調 or 競合数分待って再度押す。直らなければ連絡

※ 同期でエラーが出たまま作業を続けない。自己流で git をいじらず、画面を見せて相談してください。
